Log Only Errors
(recommended setting for live web sites)

Log All Activity
To be used only when troubleshooting.

This setting is mainly used by the DAP Support Team if required to troubleshoot your setup during installation. Set this to Log Only Errors at all times - except during initial setup and testing, when it should be set to Log All Activity, which will help during troubleshooting.

These activity/error logs can be viewed by going to "System > Logs"
