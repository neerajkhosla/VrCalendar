If you set this to "Y", then ALL WordPress Administrators (only those with "WP Admin" privileges) will automatically be logged in as DAP Admin too when they log in as WP Admin.

This means that if you have setup in-house or outsourced employees to be a WP admin to help you with your content and set up of your blog, they will now be logged in as DAP Admin as well, and will be able to perform all DAP Admin functions.
