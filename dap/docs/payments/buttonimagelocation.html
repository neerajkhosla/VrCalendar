Enter the URL to your image file.

This is the button that will be used for the buy button, and this is what will be seen on your sales page where you insert the DAP-generated button code.

If the image is in the root of your site in a folder called images, and if the name of the file is buybutton.jpg, then enter either...

a) /images/buybutton.jpg (relative url)

b) http://YourSite.com/images/buybutton.jpg (absolute url)
