* If you selected button type "Add To Order Upsell Button", then this field should contain the URL of the page you want to take the buyer to, if they decide to purchase the upsell product, and the upsell purchase completes successfully.

* If the button type is "No, Thanks Button", then this field should contain the URL of the page to which you want to take the users that do not want to purchase this upsell product.

NOTE 1: These can be the full URL of a regular Wordpress Page or Post.

NOTE 2: If you want to offer more upsells, enter the full URL of the next upsell's Sales page
