If your file has been uploaded directly under the main bucket (that you configured in the plugin's settings), then just enter JUST the file name here.

File name may NOT contain spaces, special characters, etc. Hyphens are ok. Like my-facebook-secrets.mp4

Eg: facebooksecrets.mp4

If you have created a sub-folder within your main bucket, then enter sub-foldername/filename

Eg: videos/facebooksecrets.mp4


WRONG:

You should NOT enter an entire URL - like http://yourbucket.s3.amazonaws.com/yourfilename.pdf

RIGHT:

Enter just the file name - yourfilename.pdf or your-file-name.mp4
